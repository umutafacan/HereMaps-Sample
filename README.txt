### HelloMap ###

===========================================================================
DESCRIPTION:

Simple sample App to display an NMAMapView. The project includes all
dependencies required by NMAKit.framework.

===========================================================================
INSTRUCTIONS:

1) Leave NMAKit.framework in the framework folder provided in the package. 
   Path must be HelloMap/../../framework/. 
2) Change the app bundle identifier from "com.yourcompany.HelloMap" to the
   bundle identifier of your app.
3) Update 'HelloMapAppDelegate.m' with the the app id, app code and license
   key assigned to you by your HERE representative. Note that you must
   use a license key generated for the bundle identifier you entered
   in step #2 above.
4) Enable "Automatically manage signing" or select an eligible provisioning
   profile in "General" tab

===========================================================================
BUILD REQUIREMENTS:

* Xcode 8 & iOS 8 SDK or above
* NMA SDK Version 3.0 or above

===========================================================================
TARGET PLATFORM:

* iOS 8.x & iOS 9.x
